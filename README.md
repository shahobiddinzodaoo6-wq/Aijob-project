**AIJob** is an innovative recruitment ecosystem designed to streamline the hiring process using Artificial Intelligence. Built with a premium **Glassmorphism** design and cutting-edge web technologies.

---

## ✨ Key Features

### 🤖 AI-Driven Intelligence
* **CV Analysis:** Automated evaluation of resumes for better ranking.
* **Skill Gap Identification:** Personalized insights on missing professional skills.
* **AI Cover Letters:** Instant generation of tailored cover letters and messages.

### 💼 Recruitment & Management
* **Job Lifecycle:** Full workflow for creating, editing, and managing job postings.
* **Company Profiles:** Dedicated space for organizations to manage branding and team members.
* **Advanced Dashboard:** Real-time monitoring of applications and professional statistics.

### 🤝 Networking & Localization
* **Social Feed:** Connect with professionals and share industry updates.
* **Tajik Localization (🇹🇯):** Fully translated UI optimized for the local market in Tajikistan.

---

## 🛠 Tech Stack

| Component | Technology |
| :--- | :--- |
| **Framework** | Next.js 15 (App Router) |
| **Language** | TypeScript |
| **State Management** | TanStack Query & Zustand |
| **UI/UX** | Tailwind CSS (Glassmorphism), Radix UI, Lucide Icons |
| **Data Fetching** | Axios with Proxy Integration |
| **Performance** | Next.js Turbopack |

---

## 🎨 Design Philosophy
The project implements a **Premium Glassmorphism** UI, ensuring a modern, sleek, and professional user experience with smooth animations and high-performance rendering.
